MoodTrack U

Aplicación web enfocada en el bienestar emocional y el seguimiento de hábitos en estudiantes universitarios. Permite registrar estados de ánimo, monitorear rutinas diarias y visualizar el progreso mediante métricas claras.

Descripción

MoodTrack U busca apoyar la autogestión del bienestar personal a través de una interfaz simple y funcional. El sistema permite a los usuarios identificar patrones emocionales y hábitos, facilitando la toma de decisiones para mejorar su calidad de vida.

El proyecto forma parte del desarrollo de un prototipo funcional basado en un MVP previamente validado con usuarios.

Instalación

Clonar el repositorio:

git clone https://github.com/tu-usuario/moodtrack-u.git

Entrar al proyecto:

cd moodtrack-u

Instalar dependencias:

npm install

Ejecutar en entorno local:

npm start
Uso
Crear una cuenta o iniciar sesión
Registrar estado emocional diario
Añadir y gestionar hábitos
Consultar el dashboard para ver progreso
Revisar recomendaciones generadas
Arquitectura

El proyecto sigue un enfoque basado en componentes, separando la lógica de presentación, servicios y utilidades. Se estructura para permitir escalabilidad y facilitar la integración futura con backend real.

Estado del proyecto

Prototipo funcional en desarrollo. Algunas funcionalidades operan con datos simulados (mock) mientras se define la integración completa con backend.
